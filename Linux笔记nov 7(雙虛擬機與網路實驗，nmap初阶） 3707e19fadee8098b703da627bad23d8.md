## **雙虛擬機與網路實驗 新學**

| 主题 | 命令/配置 | 说明 |
| --- | --- | --- |
| 手动添加 IP（不同子网） | sudo ip addr add 192.168.2.10/24 dev ens33 | 给网卡临时添加一个不同子网的 IP |
| 手动添加路由 | sudo ip route add 192.168.2.0/24 via 192.168.244.129 | 告诉机器访问 192.168.2.0/24 网段应走的下一跳 |
| 端口扫描（不做 ping 探测） | nmap -Pn 192.168.244.129 | 从外部扫描主机开放端口 |
| 端口 + 服务版本扫描 | nmap -Pn -sV 192.168.244.130 | 扫描端口并显示服务版本号 |
